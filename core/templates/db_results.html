Für Ihre Immobilie gibt es dabei drei Möglichkeiten:

Einfach Laden
Es wird kein Lastmanagement installiert und Sie laden Ihr Elektroauto zu jeder Zeit unkontrolliert mit voller Leistung.

Intelligent Laden
Es wird ein Lastmanagement verbaut, welches die verfügbare Ladeleistung auf die Wallboxen verteilt und den Netzanschluss vor Überlastung schützt.

Netzanschlusserweiterung
Es muss eine Netzanschlusserweiterung durchgeführt werden, da ein Lastmanagement nicht ausreicht, um den Strombedarf intelligent zu verteilen.

Für das Laden von Elektroautos zuhause kann zwischen unterschiedlichen Stromtarifen gewählt werden:

Haushaltsstrom
Der von Ihrem Elektroauto geladene Strom wird zusammen mit den anderen Haushaltsgeräten über einen Stromzähler verrechnet.

Autostrom
Der Strom für das Elektroauto wird über einen zweiten Zähler separat vom Haushalt abgerechnet.

Dynamische Stromtarife
Hier können Sie Ihr Auto laden, wenn der Strom besonders günstig ist und so von den Preisschwankungen profitieren.
